# 😸 Pumpkin's Purrfect Meme Picker

A fun and interactive meme picker built with **HTML, CSS, and JavaScript**. Choose an emotion, optionally show only animated GIFs, and enjoy a random cat meme in a polished modal.

## 📸 Preview

![Project Preview](./preview.png)

## ✨ Features

- Select an emotion using the radio buttons
- Filter results to show only animated GIFs
- Display a random meme that matches the selected mood
- Responsive layout for mobile, tablet, and desktop
- Clean modal popup with a centered design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Google Fonts (Karla)

## 🚀 How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
